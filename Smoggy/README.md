# Smoggy Project

This repository hosts the Smoggy modeling effort. Structure is organized for full reproducibility:

```
Smoggy/
  data_docs/
  preprocessing/
  feature_engineering/
  models/
  training/
  inference/
  evaluation/
  configs/
  notebooks/
  reports/
  environment.yml
  README.md
```

Each top-level directory contains code, scripts, or documentation relevant to that stage of the workflow. See `configs/` for run configurations, and `reports/` for generated outputs.
