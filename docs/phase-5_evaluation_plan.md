# Phase 5 — Evaluation Plan

Purpose: Define metrics, thresholds, and validation protocol consistent with clinical utility.

- Primary Metrics
  - ROC-AUC/PR-AUC; sensitivity @ fixed specificity; calibration (ECE)
- Subgroup Analysis
  - Age, sex, site, comorbidities
- Robustness
  - Stress tests, temporal/site shift, missingness
- Interpretability
  - SHAP/ICE summaries; feature attributions bounded by plausibility
- Reporting
  - Model card; validation report draft (reports/)
